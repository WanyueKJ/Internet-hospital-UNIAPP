### 项目介绍 

万岳科技互联网医院系统，包括与HIS对接，智慧医院、远程诊疗、网上药房、电子处方流转等子系统，满足各级别医院开展互联网诊疗


* 提供全套互联网医院源码
* 可对系统进行功能定制或二次开发
